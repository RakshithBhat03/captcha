# Captcha Signup

This is a frontend assignment for Roc8 Moonshot.

Live link : [Captcha Signup](https://captcha-kappa.vercel.app/)
## Setup

Clone the project

```bash
git clone https://github.com/RakshithBhat03/captcha
```
Go to the directory
```
cd captcha
```
Install Dependencies
``` 
npm install
```
Start Development Server
```
npm run dev
```
